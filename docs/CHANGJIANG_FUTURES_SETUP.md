# 长江期货真实行情与CTP接入清单

## 官方确认信息

长江期货官网软件中心列出了CTP综合交易平台、程序化交易终端和API对接服务。
仿真开发环境公开信息为：

- Broker ID：`4300`
- 交易前置：`tcp://61.183.150.142:20990`
- 行情前置：`tcp://61.183.150.142:20992`
- API版本：`V6.7.9`

官网同时说明，进行程序化交易前应先完成程序化交易信息报告。个人客户可通过
“长江期货APP－掌厅－程序化交易报告申请”办理；实际权限、AppID和AuthCode
应向账户所属部门或客服确认。

官方页面：<https://www.cjfco.com.cn/main/softwarecenter/index.shtml>

## 三年15分钟历史行情

官网当前列示的历史行情服务范围：

- 大商所：所有用户可申请。
- 郑商所：所有用户可申请。
- 中金所：仅限机构客户。
- 上期所、能源中心、广期所：当前页面标注暂未提供。

因此完整国内主流品种回测需要组合数据来源：先向长江期货申请大商所、郑商所
历史行情；其他交易所从长江期货交易终端导出，或使用具有授权的分钟数据平台。

最低数据要求：

- 2023-07-13至2026-07-13的15分钟K线。
- 字段包括时间、开盘、最高、最低、收盘、成交量、持仓量。
- 优先提供实际合约和主力换月映射；主连数据必须说明复权与换月方式。
- 夜盘必须归属正确的期货交易日。

## 安全配置

复制 `configs/api.changjiang.example.json` 为本机配置。不要把账号、密码、AppID
或AuthCode写入JSON；使用模板中列出的环境变量。模板默认 `enabled=false`，在
程序化报告、仿真权限和SDK适配完成前不得开启自动下单。

正式顺序：历史数据回测 → 行情只读连接 → 仿真下单 → 成交/持仓核对 → 小资金实盘。
