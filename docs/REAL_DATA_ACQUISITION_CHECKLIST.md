# 三年国内期货真实行情申请与验收清单

本清单用于向长江期货、授权数据供应商或终端服务商申请可审计历史行情。它不需要
提供交易密码、AppID、AuthCode 或任何账户敏感信息。

## 可直接发送的申请文字

> 你好，我需要用于量化策略研究和仿真回测的国内期货历史行情。请提供或说明可获取
> 的数据范围：2023-07-14 至 2026-07-14、5分钟或15分钟K线，字段包括时间、开高低收、
> 成交量、持仓量；优先提供实际合约数据及主力合约换月映射。若提供主力/连续合约，
> 请同时提供换月规则、价格调整（不复权/前复权/后复权/价差调整）说明。请说明数据
> 来源、授权范围、导出格式、费用、更新频率，以及夜盘交易日归属规则。

## 最低数据要求

| 项目 | 要求 |
| --- | --- |
| 时间 | 2023-07-14 至 2026-07-14，连续三年 |
| 粒度 | 优先5分钟（项目可安全聚合为15/30/60分钟），或原生15分钟 |
| 必填字段 | `datetime,open,high,low,close,volume,open_interest` |
| 品种 | 先覆盖 RB、HC、CU、AL、AU、AG、I、M、Y、P、PP、TA、MA、CF、SR、SA、IF、SC、LC 等主流品种 |
| 合约语义 | 实际合约文件 + 换月映射优先；主力/连续必须附构造规则 |
| 夜盘 | 明确夜盘属于哪个交易日，周五夜盘与节假日前后规则必须可解释 |
| 文件格式 | CSV/Parquet 均可；CSV 推荐 UTF-8，时间升序，一品种一文件 |

## 交易所来源边界

- 大商所、郑商所：可优先向长江期货咨询历史行情申请。
- 上期所、能源中心、广期所：当前长江期货公开页面未承诺直接提供三年历史下载；
  螺纹、黄金、铜、原油、碳酸锂等品种可能需要授权终端或数据供应商导出。
- 中金所：历史数据权限可能与客户类型有关，应单独确认股指/国债期货的可用范围。

## 收到数据后的验收

1. 保留供应商原始文件和书面说明，不要覆盖原件。
2. 导入到项目标准 CSV 后运行：

   ```powershell
   $env:PYTHONPATH = "src"
   python -m futures_quant validate-history `
     --history-dir data\real_domestic_15m `
     --suffix _15m.csv `
     --output reports\real_data_quality.csv
   ```

3. 再检查每个品种的首尾日期、交易时段缺口、重复K线、换月跳空、涨跌停和成交量异常。
4. 只有每个入选品种覆盖三年、换月规则可追溯后，才运行训练/验证/最终测试隔离和成本压力测试。

## 目录建议

```text
data/
  raw_vendor_2023_2026/         # 原始文件，只读保留
  real_domestic_15m/            # 映射后的标准CSV
reports/
  real_data_quality.csv
  data_provenance/
```

不要把交易账户、密码、AppID、AuthCode、API Token 放入这些数据目录或上传到项目仓库。
